# Nu Language 交互式演示画廊

## 📖 概述

这是一个全新的滚动百叶窗（Blinds Banner）风格的交互式演示画廊，整合了所有 Nu Language 的演示页面，提供流畅的导航体验和自动播放功能。

## 🎯 文件说明

- **`index.html`** - 主画廊页面（滚动百叶窗banner）
- `indexdemo.html` - 基础概念演示
- `index-ai-demo.html` - AI助手演示
- `index-vscode-demo.html` - VSCode集成演示
- `index-full-demo.html` - 完整功能演示

## ✨ 核心特性

### 1. 滚动百叶窗效果
- **垂直滚动动画**：页面通过垂直缩放和平移实现百叶窗打开/关闭效果
- **流畅过渡**：使用 CSS cubic-bezier 缓动函数，提供自然的动画体验
- **三态管理**：active（当前）、above（上方折叠）、below（下方折叠）

### 2. 多种导航方式

#### 🖱️ 鼠标操作
- **导航按钮**：点击 ↑/↓ 按钮切换页面
- **页面指示器**：点击底部的圆点快速跳转
- **鼠标滚轮**：滚动鼠标滚轮切换页面

#### ⌨️ 键盘快捷键
- `↑` / `Page Up` - 上一页
- `↓` / `Page Down` - 下一页
- `Space` - 切换自动播放
- `Home` - 跳转到第一页
- `End` - 跳转到最后一页

#### 📱 触摸手势（移动端）
- **向上滑动** - 下一页
- **向下滑动** - 上一页

### 3. 自动播放功能
- **循环播放**：自动按顺序切换所有演示页面
- **间隔时间**：每8秒自动切换一次
- **智能暂停**：手动切换页面时自动重置计时器
- **循环模式**：播放到最后一页后自动回到第一页

### 4. 视觉反馈
- **页面标题提示**：切换时顶部显示当前页面标题（2秒）
- **活跃指示器**：当前页面的圆点变为橙色长条
- **按钮禁用**：到达边界时相应导航按钮变暗
- **自动播放状态**：播放中的按钮有脉冲动画
- **键盘提示**：页面加载后显示操作提示

## 🎨 设计亮点

### 视觉风格
- **深色主题**：与 VSCode 编辑器风格一致
- **橙色强调色**：Nu Language 品牌色 #ff8800
- **毛玻璃效果**：控制栏使用 backdrop-filter 实现模糊背景
- **阴影和发光**：活跃元素有橙色发光效果

### 动画细节
- **百叶窗转场**：0.8秒的缓动动画
- **按钮交互**：hover 时放大，active 时缩小
- **脉冲效果**：自动播放时的呼吸动画
- **加载动画**：页面加载时的旋转 spinner

### 响应式设计
- **移动端优化**：触摸手势支持
- **自适应布局**：控制栏在小屏幕上自动缩小
- **字体缩放**：在小屏幕上自动调整字号

## 🚀 使用方法

### 快速开始
1. 使用浏览器打开 `index.html`
2. 等待加载完成（约1.5秒）
3. 使用任意导航方式浏览演示

### 推荐浏览流程
1. **第一页**：了解 Nu Language 的基本概念
2. **第二页**：体验 AI 辅助开发

4. **第四页**：观看实时转换演示

### 自动播放模式
- 点击底部的 ▶ 按钮启动自动播放
- 系统将每8秒自动切换到下一页
- 播放完最后一页后自动回到第一页
- 随时按空格键或点击 ⏸ 暂停

### 演示技巧
- **快速跳转**：直接点击页面指示器圆点
- **循环查看**：开启自动播放，无需手动操作
- **精细控制**：使用键盘快捷键快速切换
- **移动端**：使用手指上下滑动切换页面

## ⚙️ 自定义配置

### 修改自动播放间隔
在 index.html 中找到 CONFIG 对象，修改 autoplayInterval 值（单位：毫秒）

### 添加新页面
1. 创建新的演示 HTML 文件
2. 在 index.html 中添加新的 blind-panel div
3. 添加对应的指示器圆点
4. 更新 CONFIG.totalPages 值

### 修改视觉样式
- **品牌色**：搜索 #ff8800 替换为你的品牌色
- **背景色**：修改 #0d1117 和 #1e1e1e
- **动画时长**：调整 CSS 中的 transition 值
- **控制栏位置**：修改 .nav-controls 的 bottom 值

## 🐛 故障排除

### 页面无法切换
- 打开浏览器开发者工具查看错误
- 确保所有 HTML 文件在正确位置
- 清除缓存并强制刷新（Ctrl+F5）

### iframe 内容不显示
- 确保通过 HTTP 服务器访问，不要直接打开文件
- 检查 iframe 的 src 属性路径是否正确
- 确保 CSS/JS 文件路径正确

### 自动播放不工作
- 确认自动播放按钮是否激活
- 某些浏览器可能限制自动播放
- 打开控制台检查是否有 JavaScript 错误
