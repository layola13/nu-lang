# Change Log

All notable changes to the "nu-lang" extension will be documented in this file.

## [0.0.1] - 2024-12-24

### Added
- 初始版本发布
- 完整的 Nu v1.5.1 语法高亮支持
- 支持所有 Nu 关键字：`b`, `c`, `wh`, `l`, `v`, `a`, `u`, `t`, `M`, `L`, `?`, `~`
- 支持定义关键字：`S`, `E`, `TR`, `I`, `D`, `F`, `f`, `C`, `ST`, `EXT`
- 支持类型系统：`Str`, `str`, `V`, `O`, `R`, `A`, `X`, `B`, `W` 和所有原始类型
- 支持操作符：`<`, `>`, `<=`, `>=`, `==`, `!=`, `@`, `@@`, `!`
- 支持属性：`#D`, `#I`, `#?`, `#T`, `#!`, `#[...]`
- 智能括号匹配和自动补全
- 注释支持（行注释和块注释）
