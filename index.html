Why Nu?

Designed for the modern developer who values efficiency without sacrificing safety.

High Density Syntax

Compress definitions (`struct` â†’ `S`) and control flow (`return` â†’ `<`) while preserving the semantic core. Optimized for LLM context windows and developer typing speed.

Bidirectional Transpilation

Lossless conversion between Rust and Nu. Use `rust2nu` to modernize legacy codebases and `nu2rust` to compile standard binaries.

First-Class Tooling

Production-ready VSCode extension with Auto-Compilation, Error Mapping, F5 Debugging, and real-time formatting.

Complete Toolchain

Seamless bidirectional conversion between Rust, Nu, and TypeScript ecosystems.

rust2nu

Convert Rust source code to Nu high-density syntax. Modernize legacy codebases instantly.

rust2nu input.rs output.nu

nu2rust

Compile Nu code back to standard Rust. Full compatibility with cargo build system.

nu2rust main.nu main.rs

cargo2nu

Convert entire Cargo projects to Nu projects. Supports workspaces and build scripts.

cargo2nu my_project nu_project

nu2cargo

Generate Cargo projects from Nu projects. Seamless integration with Rust ecosystem.

nu2cargo nu_project cargo_project

nu2ts

Transpile Nu to TypeScript. Bridge systems programming with web development.

nu2ts app.nu app.ts

VSCode Extension

Full IDE support with syntax highlighting, auto-compilation, debugging, and error mapping.
